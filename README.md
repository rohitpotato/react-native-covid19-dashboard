# Features
1. Area Charts for displaying stats.
2. Country Selection.
3. COVID-19 News Articles.

# Installation
1. ``` yarn install ``` to install all the dependecies. 
2. ``` react-native run-android ``` to run on android.
3. ``` react-native run-ios``` to run on ios.
